ISTRUZIONI PER L'UTILIZZO DELL'APPLICAZIONE MUSEO MUPIN

PER CONNETTERE IL DATABASE MYSQL:
-Creare un DB mySql su phpMyAdmin
-Configurare le variabili d'ambiente nel .env:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1                           //configurare url
DB_PORT=3306                                //inserire la propria porta
DB_DATABASE=mupin_esame                     //inserire nome del proprio DB
DB_USERNAME=root                            //inserire username
DB_PASSWORD=                                //inserire password

-install composer
-npm install
-npm run dev (per attivare i template, deve essere attivo durante l'utilizzo dell'applicazione web)
-php artisan serve (per avviare il server in localhost)

PER POPOLARE IL DATABASE:
-php artisan migrate (crea le tabelle con relative colonne nel DB mySql)
-php artisan db:seed (popola il database mySql con un insieme di record)
